MyFirst-HexoBlog

我的个人Blog——Hexo搭建（windows）

前置准备

Node.js
Git

搭建本地blog框架

  • 因npm服务器在国外安装为了防止下载时速度过慢，先安装cnpm镜像
    淘宝NPM
    npm install -g cnpm –registry=https://registry.npm.taobao.org

  • 使用cnpm安装hexo模块
    cnpm install -g hexo-cli –save

  • 新建一个空文件夹，在该路径下对hexo初始化
    hexo init
    时间较长且有过进度无响应状况，尝试多次中断重装即可

  • 初始化完成后启动服务：hexo s

推送至远端（Github）

  • 安装模块
    cnpm install –save hexo-deployer-git

  • 配置_config.yml文档：
    Deployment
    Docs: https://hexo.io/docs/deployment.html
    deploy:
    type: git
    repo: https://github.com/JamesZhu997/JamesZhu997.github.io.git（地址为Github上新建仓库后的地址）
    branch: master

  • 开始向远端推送：
    hexo d
    可能会报错：Unable to auto-detect email address……
    解决方法如下:
    输入git config –global user.name”Your Name”

    git config --global user.email"Your email"

    重新hexo d

成功后访问https://JamesZhu997.github.io.git
